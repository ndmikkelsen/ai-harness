# Phase Workspace

Create one folder per phase under `.planning/phases/<phase-slug>/`.

Recommended structure:

```text
.planning/phases/<phase-slug>/
  CONTEXT.md
  RESEARCH.md
  PLAN.md
  SUMMARY.md
  VERIFICATION.md
  UAT.md
```

If a phase needs multiple plans, nest them under the phase folder so each plan can keep its own `PLAN.md` and `SUMMARY.md`.
